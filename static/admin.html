系统说明
  • 默认数据库为内置 SQLite，可在验证成功后切换到 MySQL。
  • 数据库体积达到限制值后，爬虫会自动停机，避免继续入库。
  • 标题关键词、最小/最大大小过滤会在保存磁力时即时生效。
  • 高 CPU 配置下将自动提升 metadata 调度强度，但仍受网络环境与目标节点响应影响。
